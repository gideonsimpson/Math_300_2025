# Math 300 - Fall 2025

Course codes and materials for Math 300, Fall 2025. This repository will be populated with lecture materials, assignments, projects, and examples throughout the term.

## Repository Structure

```
Math_300_2025/
├── lectures/          # Lecture materials and in-class demonstrations
├── homework/          # Homework assignments and starter code
├── projects/          # Course projects and extended assignments
├── examples/          # Standalone examples and reference implementations
├── utils/             # Shared utilities and helper functions
├── docs/              # Course documentation and guides
├── README.md          # This file
└── LICENSE            # MIT License
```

## Getting Started

1. **Prerequisites**: This course uses Julia for computational work. Install Julia from [julialang.org](https://julialang.org/).

2. **Repository Updates**: This repository will be updated regularly throughout the term with new materials, so make sure to pull the latest changes frequently.

3. **Organization**: Materials are organized by type (lectures, homework, projects) and will be added as the course progresses.

## Directory Descriptions

- **`lectures/`**: Contains code examples, demonstrations, and materials used during class sessions
- **`homework/`**: Homework assignments with starter code and instructions
- **`projects/`**: Larger projects and extended assignments
- **`examples/`**: Self-contained examples and reference implementations
- **`utils/`**: Shared utility functions and helper modules used across the course
- **`docs/`**: Course documentation, setup guides, and reference materials

## Usage

Each directory contains its own README with specific information about the contents and organization. As materials are added throughout the term, check the individual directories for the latest assignments and resources.

## Contributing

This is a course repository managed by the instructor. Students should follow assignment submission guidelines provided in class.
