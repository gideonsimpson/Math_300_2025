# Homework

This directory contains homework assignments, starter code, and solutions.

## Structure
- Each homework assignment has its own subdirectory (e.g., `hw_01/`, `hw_02/`)
- Starter code and templates
- Solution files (when available)
- Instructions and requirements

## Naming Convention
- Use `hw_XX/` format for homework directories
- Include descriptive names: `hw_01_linear_algebra/`
- Separate student and instructor materials as needed
