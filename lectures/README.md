# Lectures

This directory contains lecture materials, code examples, and demonstrations used during class sessions.

## Structure
- Each lecture may have its own subdirectory (e.g., `lecture_01/`, `lecture_02/`)
- Code files (.jl for Julia)
- Supporting materials (data files, images, etc.)
- Notes and documentation

## Naming Convention
- Use descriptive names for lecture directories
- Include lecture number for easy ordering
- Example: `lecture_01_introduction/`, `lecture_02_linear_systems/`
